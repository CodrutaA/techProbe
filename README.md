# techProbe

![Tests](https://github.com/CodrutaA/techProbe/actions/workflows/test.yml/badge.svg)

## Install project

```bash
npm install
```

## Start server

```bash
npm start
```

## Tests

- [API tests](./tests/api/)

```bash
npm run test:api
```

- [Playwright tests](./tests/e2e/)

```bash
npm run testplaywright:test
```

- [Cypress tests](./cypress/e2e/)

```bash
npm run cy:run
```
